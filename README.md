| Date | 12th September 2022|
| - | - |
- Website is [Live!](https://google-forms-dupe.vercel.app/):

```bash
https://google-forms-dupe.vercel.app/
```
## Setup:
```bash
git clone https://github.com/frostedCupcake/Google-Forms-Dupe.git
cd .\Google-Forms-Dupe\
npm install
npm run start
```
